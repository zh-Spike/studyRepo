### 题目
实现一个 Trie (前缀树)，包含 insert, search, 和 startsWith 这三个操作。

示例:
```
Trie trie = new Trie();

trie.insert("apple");
trie.search("apple");   // 返回 true
trie.search("app");     // 返回 false
trie.startsWith("app"); // 返回 true
trie.insert("app");   
trie.search("app");     // 返回 true
```
说明:

- 你可以假设所有的输入都是由小写字母 a-z 构成的。
- 保证所有输入均为非空字符串。

### 思路

前缀树就是 ~~有着相同前缀的的树~~

一般的 这里是 26 个字母 用数组会比较方便 如果是 6w 个汉字 ? 可用 HashMap

### Code
```java
// 定义结构体
class TrieNode {
    // 通过该节点的次数
     int pass;
     // 在该节点结束的次数
     int end;
    // HashMap<Char, Node> nexts;
    // TreeMap<Char, Node> nexts;
    // 指向下一个的路径方向 这里是 26 个字母
    TrieNode[] nexts;

    TrieNode() {
        pass = 0;
        end = 0;
        // nexts[0] == null 没有走向‘a’的路
        // nexts[0] != null 有走向‘a’的路
        // ...
        // nexts[25] != null 有走向‘z’的路
        nexts = new TrieNode[26];
    }
}

    class Trie {
        /**
         * Initialize your data structure here.
         */
        private TrieNode root;

        public Trie() {
            // 实例化
            root = new TrieNode();
        }

        /**
         * Inserts a word into the trie.
         */
        public void insert(String word) {
            // 判空
            if (word == null) {
                return;
            }
            // String 转 字符数组
            char[] chs = word.toCharArray();
            TrieNode node = root;
            // root的通过数++
            node.pass++;
            // 看走第几条路
            int index = 0;
            for (int i = 0; i < chs.length; i++) {
                // 算第几条路
                index = chs[i] - 'a';
                // 如果当前路径是第一次走 创建这条路
                if (node.nexts[index] == null) {
                    node.nexts[index] = new TrieNode();
                }
                // 推到下一个点
                node = node.nexts[index];
                // 走到点的通过数++ 
                node.pass++;
            }
            // 全都走完了 结尾点++
            node.end++;
        }

        /**
         * Returns if the word is in the trie.
         */
        public boolean search(String word) {
            if (word == null) {
                return false;
            }
            char[] chs = word.toCharArray();
            TrieNode node = root;
            int index = 0;
            for (int i = 0; i < chs.length; i++) {
                index = chs[i] - 'a';
                // 看看这条路有没有走过 没走过 直接 false
                if (node.nexts[index] == null) return false;
                node = node.nexts[index];
            }
            // 因为他是前缀树 所以不可能出现中间段存在 开头结尾
            // node.end 就是它从头到尾回复的这个词出现的频率
            return node.end>0?true:false;
        }

        /**
         * Returns if there is any word in the trie that starts with the given prefix.
         */
        public boolean startsWith(String prefix) {
            if (prefix == null) {
                return false;
            }
            char[] chs = prefix.toCharArray();
            TrieNode node = root;
            int index = 0;
            // 判断过程基本同 search
            for (int i = 0; i < chs.length; i++) {
                index = chs[i] - 'a';
                if (node.nexts[index]==null)return false;
                node = node.nexts[index];
            }
            // 他这个 prefix <= word 所以不用看 node.end  
            return true;
        }
    }
```
