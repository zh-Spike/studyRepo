## VIM
1. 多行编辑 
    1. ctrl + V 
    2. jj
    3. I
2. 剪切板寄存器
    1. 剪切板 "+y 或 "*y
    2. "+p 粘贴系统剪切板内容到当前位置后
    ```
    ""     默认unnamed寄存器，最近一次"d","c","s","x","y"复制，删除，修改内容
    "0     最近一次"y"复制内容
    "1     最近一次"d","c","s","x"删除，修改内容
    "2     上一次"d","c","s","x"删除，修改内容
    "3     上上次"d","c","s","x"删除，修改内容
    ...     
    "9     [1-9]数字以此类推
   
    "a     字母寄存器，供用户指定使用，比如"ay就是复制到"a寄存器
    "b
    ...
    "z
    
    "-     少于一行的"d","c","x"删除内容
    ".     只读寄存器
    ":     只读寄存器
    "%     只读寄存器
    "#     只读寄存器

    "+     映射系统剪贴板 (有的默认设置不支持)
    "*     映射系统剪贴板 (有的默认设置不支持)
    ```
